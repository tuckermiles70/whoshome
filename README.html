Who's Home

About

Who's home is a tool for polling your home's local network and sending an alert whenever someone arrives. Simply install the necessary components, configurate as shown below, and you're off!

Setup

Your computer must have the Nmap security Scanner for Linux/Mac which you can download here. Next, create a file names "config.py" as below.
